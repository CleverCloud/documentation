What's new Clever Cloud ? Here are the major improvments released to our public products and interfaces.

## 2024-01-02
- Pulsar 3.2.0 is deployed and used on our platform

## 2023-12-24
- A changelog in the documentation

## 2023-12-22
- Sozu can now be used in front of any add-on

## 2023-12-20
- Better filtering and performances on Sozu services

## 2023-12-19
- Links with an icon to the new documentation in [the Console](https;//console.clever-cloud.com)
- Review App Action published [on the GitHub Marketplace](https://github.com/marketplace/actions/clever-cloud-review-app-on-pull-requests)
- Clever Components v12 [are available](https://github.com/CleverCloud/clever-components/releases)

## 2023-12-18
- New IP for DNS in PAR region are shown in [the Console](https;//console.clever-cloud.com)
- PHP 8 is used for new PHP applications
- Java 21 is used for new Java applications

## 2023-12-15 
- Sozu 0.15.18 is available and deployed on our services
- Clever Cloud's logo is available from [Simple icons](https://simpleicons.org/)

## 2023-12-13
- Cellar migration tool 2.10 [is available](https://github.com/CleverCloud/cellar-migration/releases)

## 2023-12-11
- Our Vault product enters private alpha stage

## 2023-12-08
- The `direct-host-only` option is available for MongoDB, MySQL and PostgreSQL through [Clever Tools](https://github.com/CleverCloud/clever-tools)

## 2023-12-07
- A toast in [the Console](https;//console.clever-cloud.com) tells user to update DNS configuration when an application is deployed in another region
- We've improved our Cellar capacity, with 1 replica per datacenter for all objects

## 2023-11-29
- You can now check if an application is a task through its `instance.lifetime` value in [the API](../openapi/)

## 2023-11-27
- Deploy Ollama with a chat interface on Clever Cloud from [this repository](https://github.com/CleverCloud/ollama-chat-deploy)
- Sozu 0.15.17 is available and deployed on our services with improved performances

## 2023-11-24
- The new Clever Cloud documentation is available online
